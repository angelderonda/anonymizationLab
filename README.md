## INSTALLATION ⚙️

In Windows:
```
python -m venv env
.\env\Scripts\activate.bat      #Command prompt
.\env\Scripts\Activate.ps1      #Powershell
pip install -r requirements.txt
```

In Linux:
```
python -m venv env
./env/Scripts/activate
pip install -r requirements.txt
```

## APPLICATION 🌐

To launch the application: 
```
cd .\src\
python .\app.py
```
